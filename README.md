# pwn.college Writeups — Pedro Tuttman

Repositório com writeups dos desafios de [pwn.college](https://pwn.college), plataforma de aprendizado em segurança ofensiva com foco em exploração de binários.

Cada writeup documenta o processo completo de análise e exploração — das ferramentas utilizadas até o raciocínio por trás de cada decisão.
